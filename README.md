# 机器学习课程作业@SZU
## Ex1:威斯康星州乳腺癌诊断情况预测
### 环境依赖
- Python 3.9.12
- numpy == 1.21.5
- pandas == 1.4.2
- lightgbm == 3.3.2
- scikit-learn == 1.0.2

### 目录结构
```
./Ex1
├── data.csv, csv格式的训练数据
├── run_lgb.py, 训练梯度提升决策树模型
├── run_lr.py, 训练逻辑回归模型
├── run_rf.py, 训练随机森林模型
├── run_svm.py, 训练支持向量分类模型
└── ...
```

### 运行流程
```
cd ./Ex1
python run_[MASK].py
```
